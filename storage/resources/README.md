Local runtime resources live here.

This directory is intentionally ignored except for this README. Copy licensed
runtime assets here so the Web project can run without depending on the legacy
desktop project directory.

Expected layout:

```text
storage/resources/
  ancient/
    ancient_background.png
    ancient_voice.wav
  fonts/
    HYShangWeiShouShu.ttf
    ZhiMangXing-Regular.ttf
    OFL.txt
  bgm_library/
    *.mp3
```

Do not commit commercial fonts, voice samples, music files, generated videos,
or user-provided media.
